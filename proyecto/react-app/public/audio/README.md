# Audio Files for EduNow

This folder contains audio files for the application.

## Required Files:
- `background-music.mp3` - Background music that plays continuously
- `button-click.mp3` - Sound effect for button clicks
- `success.mp3` - Sound for successful actions
- `error.mp3` - Sound for errors or incorrect answers

## Notes:
- All files should be in MP3 format for better browser compatibility
- Background music should be loopable
- Sound effects should be short (1-2 seconds)
- Volume levels should be balanced for good user experience
